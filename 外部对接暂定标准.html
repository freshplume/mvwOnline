活动模式对外接口文档


前言

本文档所涉及的接口均部署在医视在线测试环境。


1.1 添加培训人员

接口描述：用于培训添加培训人员，支持批量

请求方式：POST

请求URL：api/activityHuman/adds

请求参数：

名称     类型     说明     是否必须     备注

activityId     String     活动ID     是     由医视界提供

institudeId     String     机构ID     是     由医视界提供

mobile     String     手机号     是     人员批量最大100条

name     String     姓名     是    

{

  activityId:’’,        

  institudeId:’’,          

  personList:[

{

          mobile:’’,

      password:’’

},

{

          mobile:’’,

      password:’’

},

]

}

返回参数：

{

  result:’true’,           //true表示请求成功 ，false表示请求失败 

  msg:’’

}



1.2 删除人员

接口描述：用于删除培训中的单个人员

请求方式：POST

请求URL：api/activityHuman/delete

请求参数：

名称     类型     说明     是否必须     备注

activityId     String     活动ID     是     由医视界提供

institudeId     String     机构ID     是     由医视界提供

mobile     String     手机号     是    

{

  activityId:’’,        

  institudeId:’’,          

  mobile:’’

}

返回参数：

{

  result:’true’,           //true表示请求成功 ，false表示请求失败 

  msg:’’

}


1.3 学员学习

接口描述：请求验证参数后，跳转学习页面

请求方式：GET

请求URL：api/getProjectInfo?mobile={mobile}&projectId={ projectId }

请求参数：

名称     类型     说明     是否必须     备注

projectId     String     培训ID     是     由医视界提供

mobile     String     手机号     是    

返回参数：

无



1.4获取培训记录

接口描述：获取培训中全部学员学习记录

请求方式：POST

请求URL：api/getProjectUserTrainList

请求参数：

名称     类型     说明     是否必须     备注

projectId     String     培训ID     是     由医视界提供

page     String     当前页数     是    

pageSize     String     每页记录数     是     最大100

{

  projectId:’’,        

  page:’’,          

  pageSize:’’

}

返回参数：

{

result:’true’,            

  msg:’’,

total:’’,

  trainList:[

0:{

     mobile:’’,

     totalWatchTimeText: "00:00:00",      //培训总时长

     projectFinishRate: 0.0,              //培训进度，百分比

     projectScore: 0                    //培训考试分数

},

1:{......},

2:{......}

]

}
